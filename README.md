# Framing shop
Example of black-box test design applying equivalence partitioning and 3-value boundary value analysis.

## Usage
1. Create a Python virtual environment: `python -m venv venv`
2. Activate it: `venv\Scripts\activate`, `.\venv\Scripts\Activate`, `source venv/bin/activate`, depending on OS and CLI
3. Install the dependencies: `pip install -r requirements.txt`
4. Run the tests: `pytest`
